# `RunAsyncExit<A, E>`

`RunAsyncExit` allows for running an asynchronous `Effect` and mapping the result to a JSX element.

[`Effect.runPromiseExit` Documentation {"\u{21D2}"}](https://effect.website/docs/getting-started/running-effects/#runpromiseexit)

[`Effect.runPromiseExit` Reference {"\u{21D2}"}](https://effect-ts.github.io/effect/effect/Effect.ts.html#runpromiseexit)

`RunAsyncExit` runs the `Effect` as a promise and integrates with `Suspense`.
If the effect succeeds, the result is handled by `onSuccess`.
If the effect fails, the result is handled by `onFailure`.
